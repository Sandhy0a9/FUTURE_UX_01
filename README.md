# FUTURE_UX_01

## Task
Local Service Business Website Redesign

## Objective
Redesign a local beauty salon website with a modern, user-friendly, and conversion-focused UI.

## Tools Used
- Figma
- Iconify
- Unsplash

## Pages Designed
- Home Page
- Services Page
- Contact Page

## Features
- Modern Hero Section
- Service Cards
- Call-to-Action Sections
- Testimonials
- Contact Form
- Responsive Layout
- Clean and Minimal UI

## Design Rationale
The website was redesigned to improve user experience, increase customer engagement, and make appointment booking easier through a clean and visually appealing interface.
